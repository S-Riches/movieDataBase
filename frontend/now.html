Movie DB is a full stack application package that requires an internet browser and the docker engine to run. Please look a the readme to see how to set it up.

To view all movies from a certain site then go to the respective page on the navbar.

Alternatively you can go to the search page and search for a specific film and this will query the database and return what sites you can find the respective film on.
